## 新潟大学教科書フリマアプリのバックエンドリポジトリ
### 新潟大学プログラミングサークルPLOTのチーム開発プロジェクトです

### セットアップ

#### Dockerをインストール
* コマンドラインか、以下でデスクトップアプリをインストール
* https://www.docker.com/ja-jp/products/docker-desktop/


#### Dockerコンテナの起動
```
docker-compose build
```

```
docker-compose up
```

#### localhostにアクセス
```
http://localhost:3000/
```
